# SenseiGram

**Advanced Telegram Bot Manager & Client for Android**

A native Android app for managing Telegram bots completely serverless.

[![Download](https://img.shields.io/github/v/release/jubairbro/SENSEIGRAM?label=Download)](https://github.com/jubairbro/SENSEIGRAM/releases/latest)

---

## 🌟 Key Features

- **Bot Identity Editor:** Update Name, About, Bio, and Profile Photo directly from the app.
- **Fetch Users:** Get a list of users who recently interacted with your bot.
- **Send Rich Media:** Text, Photos, Videos, Documents (Device upload or URL).
- **Inline Keyboard Builder:** Support for new **Colored Buttons** (Primary, Success, Danger).
- **Format Text:** Bold, Italic, Code, URL, Spoiler natively.
- **Target System:** Save unlimited target chats or users.
- **Privacy & Settings:** Silent Mode, Protect Content, Media Spoilers, No Web Page Previews.
- **Modern UI:** True Dark/AMOLED modes with 5 distinct accent colors.

*(Includes API ID: 26158708 for future full-client TDLib expansion)*
